# Honeymoon Engine
The Honeymooner Core v1.0

Notes:
- Uses post types: destinations and packages
- Leads stored in wp_hm_leads
- REST endpoint: POST /wp-json/honeymooner/v1/leads
- CPT REST endpoints:
  /wp-json/wp/v2/destinations
  /wp-json/wp/v2/packages

If you previously installed an older plugin build using hm_package / hm_destination, deactivate and delete it first.
